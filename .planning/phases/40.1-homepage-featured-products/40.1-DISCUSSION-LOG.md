## Featured selection & ordering

### Ordering
| Option | Description | Selected |
|--------|-------------|----------|
| Newest first (date desc) | isFeatured boolean only; sort by existing `date`, no new field | |
| New featuredOrder field | Dedicated global `featuredOrder` number; exact arrangement control | ✓ |

**User's choice:** New featuredOrder field
**Notes:** displayOrder is per-category and unsuitable for a cross-category homepage grid.

**User's choice:** Cap at 8
