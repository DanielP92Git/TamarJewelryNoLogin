Implement `startBulkTranslation()`:
```javascript
function startBulkTranslation() {
  // Reset state
  bulkStats = { success: 0, failed: 0, skipped: 0 };
  bulkFailedProducts = [];

  // Show progress, hide summary
  const progressEl = document.querySelector('.bulk-progress');
  const summaryEl = document.querySelector('.bulk-summary');
  const startBtn = document.querySelector('.btn-start-bulk');
  if (progressEl) progressEl.style.display = 'block';
  if (summaryEl) summaryEl.style.display = 'none';
  if (startBtn) startBtn.style.display = 'none';

  // Reset progress UI
  updateBulkProgress(0, 0, '');
  updateBulkStats();

  // Get auth token
  const authToken = localStorage.getItem('auth-token');

  // Create EventSource with query param auth
  bulkEventSource = new EventSource(
    `${API_URL}/admin/translate/bulk?token=${encodeURIComponent(authToken)}`
  );

  bulkEventSource.addEventListener('start', onBulkStart);
  bulkEventSource.addEventListener('progress', onBulkProgress);
  bulkEventSource.addEventListener('success', onBulkSuccess);
  bulkEventSource.addEventListener('error', onBulkError);
  bulkEventSource.addEventListener('complete', onBulkComplete);
  bulkEventSource.onerror = onBulkConnectionError;
}
```

Implement SSE event handlers:
- `onBulkStart(event)`: Parse JSON, update total count display
- `onBulkProgress(event)`: Parse JSON `{ current, total, productName }`, call `updateBulkProgress(current, total, productName)`
- `onBulkSuccess(event)`: Increment `bulkStats.success`, call `updateBulkStats()`
- `onBulkError(event)`: Parse JSON, push to `bulkFailedProducts`, increment `bulkStats.failed`, call `updateBulkStats()`
- `onBulkComplete(event)`: Close EventSource, show summary with counts, show retry button if failures > 0, show toast via `showSuccessToast` or `showInfoToast`
- `onBulkConnectionError(err)`: Close EventSource, show `showErrorToast('Connection lost. Please retry.')`, show summary with partial results

Implement `cancelBulkTranslation()`:
```javascript
function cancelBulkTranslation() {
  if (bulkEventSource) {
    bulkEventSource.close();
    bulkEventSource = null;
  }
  showInfoToast('Bulk translation cancelled');
  showBulkSummary(true); // true = cancelled
}
```

Implement UI update helpers:
- `updateBulkProgress(current, total, productName)`: Update progress bar width, counter text, current product name (use textContent)
- `updateBulkStats()`: Update success/failed/skipped counters
- `showBulkSummary(cancelled = false)`: Hide progress, show summary with counts. If cancelled, show "Translation Cancelled" heading. If failures > 0, show retry button. Build failed products list with first 10 shown and expandable "Show all N" toggle.

**Important implementation notes:**
- Use `API_URL` variable (already resolved globally in BisliView.js) for the EventSource URL
- Use existing toast helpers: `showSuccessToast()`, `showErrorToast()`, `showInfoToast()`
- Follow existing admin patterns: use `clear()` to reset page content, use `pageContent.insertAdjacentHTML("afterbegin", markup)` for rendering
- ALWAYS use `textContent` when displaying product names from server (XSS prevention)
- Progress bar width: `Math.round((current / total) * 100) + '%'`
- Handle the edge case where `total === 0` (no products need translation): show immediate message "All products are already translated!" with no progress bar
