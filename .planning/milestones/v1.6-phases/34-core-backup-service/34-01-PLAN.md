Create backend/services/backupService.js with the following exported functions. Use CommonJS (require/module.exports) to match all other backend files.

FUNCTION 1 - createBackupS3Client():
Creates and returns an AWS.S3 instance using backup-specific credentials. Requires aws-sdk (already installed v2.1693.0). Constructor:
- endpoint: new AWS.Endpoint(process.env.BACKUP_SPACES_ENDPOINT)
- accessKeyId: process.env.BACKUP_SPACES_KEY
- secretAccessKey: process.env.BACKUP_SPACES_SECRET
- region: process.env.BACKUP_SPACES_REGION || undefined
Do NOT import or reuse the S3 client from index.js (it is not exported). Use BACKUP_SPACES_* credentials, not SPACES_* credentials.

FUNCTION 2 - buildBackupFilename():
Returns ISO-timestamped filename. Implementation:
  return 'backup-' + new Date().toISOString().replace(/:/g, '-') + '.archive.gz';
Per BKUP-03, colons replaced with dashes for filesystem compatibility. Lexicographic sort = chronological sort.

FUNCTION 3 - spawnMongodump(mongoUri):
Returns a Promise resolving with a Buffer containing the gzip archive.
- Use child_process.spawn (NOT execFile -- spawn avoids maxBuffer limits)
- Read mongodump path: process.env.MONGODUMP_PATH || 'mongodump' (Phase 33 D-08)
- Args array: ['--uri', mongoUri, '--archive', '--gzip'] per D-01
- stdio: ['ignore', 'pipe', 'pipe'] -- stdin ignored, stdout piped for archive, stderr piped for errors
- Collect stdout chunks into array, then Buffer.concat(chunks) in close handler (D-02)
- Collect stderr chunks separately for error reporting
- close handler: if code !== 0, reject with stderr content. REDACT MongoDB URI from stderr before using it in the error message: stderr.replace(/(mongodb[^s]?:\/\/)[^\s@]+@/g, '$1[REDACTED]@')
- error handler: reject with the error (handles ENOENT when binary not found)
- CRITICAL: Never log mongoUri directly anywhere in this file

FUNCTION 4 - runRetentionCleanup(s3, bucket, prefix, retentionCount):
Deletes surplus backups beyond retention count.
- Call s3.listObjectsV2({ Bucket: bucket, Prefix: prefix }).promise()
- Sort (listResp.Contents || []) by Key lexicographically ascending (oldest first)
- Calculate surplus: objects.slice(0, Math.max(0, objects.length - retentionCount))
- If surplus.length === 0, return 0
- Call s3.deleteObjects({ Bucket: bucket, Delete: { Objects: surplus.map(o => ({ Key: o.Key })) } }).promise()
- Check deleteResp.Errors array -- if non-empty, throw new Error with the error details (caller catches as retentionError per D-09)
- Return surplus.length
- Add comment: "listObjectsV2 returns max 1000 objects per call; with retention ~14, pagination is unnecessary"

FUNCTION 5 - runBackup():
Main orchestrator. Per D-07, returns result AND logs it. Steps:
1. const startedAt = Date.now()
2. const filename = buildBackupFilename()
3. Initialize result: { timestamp: new Date().toISOString(), status: 'success', filename, sizeBytes: null, durationMs: null, retentionDeleted: 0, error: null, retentionError: null }
4. try block:
   a. GUARD: check BACKUP_BUCKET, BACKUP_SPACES_ENDPOINT, BACKUP_SPACES_KEY, BACKUP_SPACES_SECRET are all truthy. If any missing: throw new Error('Missing BACKUP_SPACES_* credentials or BACKUP_BUCKET')
   b. GUARD: check process.env.MONGO_URL is truthy. If missing: throw new Error('MONGO_URL is not set'). NOTE: env var is MONGO_URL not DATABASE_URL (confirmed from backend/config/db.js)
   c. const buffer = await spawnMongodump(process.env.MONGO_URL)
   d. result.sizeBytes = buffer.length (per D-03)
   e. const s3 = createBackupS3Client()
   f. const prefix = process.env.BACKUP_SPACES_PREFIX || 'backups/'
   g. const key = prefix + filename
   h. await s3.putObject({ Bucket: process.env.BACKUP_BUCKET, Key: key, Body: buffer, ContentType: 'application/gzip' }).promise()
   i. Retention cleanup in nested try/catch (per D-09 -- failure does NOT change status):
      try {
        const retentionCount = parseInt(process.env.BACKUP_RETENTION_COUNT, 10) || 14;
        result.retentionDeleted = await runRetentionCleanup(s3, process.env.BACKUP_BUCKET, prefix, retentionCount);
      } catch (retErr) {
        result.retentionError = retErr.message;
      }
5. catch block: result.status = 'failed'; result.error = err.message (per D-08, no retry)
6. finally: result.durationMs = Date.now() - startedAt
7. console.log('[backup] ' + JSON.stringify(result)) (per D-06/MON-01)
8. return result (per D-07 -- Phase 35 manual trigger returns this directly)
