### Integration Issues

1. **Rate limiter header inconsistency (Minor):** `adminRateLimiter` in `routes/backup.js` omits `standardHeaders: 'draft-7'` and `legacyHeaders: false` present in the main `index.js` instance. Limit enforcement (120 req/15 min) is identical; only response headers differ. Affects ADM-01, REST-03.

2. **Binary path not shared (Low):** Phase 33's `verifyMongodumpBinary()` resolves the path at startup, but Phase 34's `spawnMongodump()` re-resolves `MONGODUMP_PATH` independently at runtime. In practice env vars are static — architectural observation only. Affects BKUP-04, BKUP-01.

3. **BackupLog barrel import unused (Cosmetic):** `BackupLog` is registered in `models/index.js` but all consumers use direct `require('../models/BackupLog')` instead of the barrel. Functional — no impact. Affects MON-03.
